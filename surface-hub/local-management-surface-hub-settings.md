# Gestion locale des paramètres SurfaceHub

Après la configuration initiale du Microsoft SurfaceHub, les paramètres de l’appareil peuvent être gérés localement par le biais de l’application **Paramètres**.

## Paramètres SurfaceHub

Les SurfaceHub disposent de nombreux paramètres communs avec d’autres appareils Windows, mais également de paramètres uniquement configurables sur les SurfaceHub. Le tableau ci-dessous répertorie les paramètres uniquement configurables sur les SurfaceHub. 

| Paramètre | Emplacement | Description |
| ------- | -------- | ----------- |
| Compte d’appareil | Surface Hub> Comptes | Définissez ou modifiez le compte d’appareil du SurfaceHub. |
| État de synchronisation du compte d’appareil | Surface Hub> Comptes | Vérifiez l’état de synchronisation du calendrier et de la messagerie du compte d’appareil sur le SurfaceHub. |
| Rotation du mot de passe | Surface Hub> Comptes | Choisissez ou non d’autoriser le SurfaceHub à faire tourner automatiquement le mot de passe du compte d’appareil.|
| Modifier le mot de passe du compte d’administrateur  | Surface Hub> Comptes | Modifiez le mot de passe du compte d’administrateur local. Ce paramètre est uniquement disponible si vous avez configuré l’appareil pour utiliser un administrateur local lors de la première utilisation. |
| Gestion des périphériques | Surface Hub> Gestion des appareils | Gérez les stratégies et les applications d’entreprise à l’aide de la gestion des périphériques mobiles (GPM). |
| Packages d’approvisionnement | Surface Hub> Gestion des appareils | Définissez ou modifiez les packages d’approvisionnement installés sur le Surface Hub. |
| Ouvrir l’application Microsoft Store | Surface Hub> Applications et fonctionnalités | L’application Microsoft Store n'est disponible que pour les administrateurs par l'intermédiaire de l’application Paramètres. |
| Nom de domaine SkypeEntreprise | Surface Hub> Appel et audio | Configurez un nom de domaine pour votre serveur Skype Entreprise. |
| Volume du haut-parleur par défaut | Surface Hub> Appel et audio | Configurer le volume du haut-parleur par défaut pour le Surface Hub lorsqu’il démarre une session. |
| Paramètres par défaut du microphone et du haut-parleur | Surface Hub> Appel et audio | Configurez un microphone et un haut-parleur par défaut pour les appels, et un haut-parleur par défaut pour la lecture de média. |
| Activer DolbyAudioX2 | Surface Hub> Appel et audio | Configurez les améliorations de haut-parleur DolbyAudioX2. |
| Ouvrir automatiquement l’application Connexion | SurfaceHub> Projection | Choisissez si la projection doit ouvrir automatiquement l’application Connexion ou attendre une entrée utilisateur avant de le faire. |
| Désactiver la projection sans fil à l’aide de Miracast | SurfaceHub> Projection | Choisissez si les présentateurs peuvent utiliser la projection sans fil sur le SurfaceHub à l’aide de Miracast. |
| Exiger un code confidentiel pour la projection sans fil | SurfaceHub> Projection | Choisissez s’il convient d’entrer un code confidentiel avant d’utiliser la projection sans fil. |
| Canal de projection sans fil (Miracast) | SurfaceHub> Projection | Définissez le canal pour la projection Miracast. |
| Informations de réunion affichées sur l’écran d’accueil | Surface Hub> Écran d’accueil | Choisissez si l’organisateur, l’heure et l’objet de la réunion apparaissent sur l’écran d’accueil. |
| Arrière-plan de l’écran d’accueil |  Surface Hub> Écran d’accueil | Choisissez une image d’arrière-plan pour l’écran d’accueil. |
| Fenêtre délai d’expiration de session pour l’accueil | Session surface Hub > & alimentation | Choisissez la durée à attendre avant que le Surface Hub ne revienne à l’écran d’accueil si aucun mouvement n’est détecté. |
| Reprendre la session | Session surface Hub > & alimentation | Choisissez si, lorsqu’aucun mouvement n’est détecté pendant un certain temps, les utilisateurs doivent être autorisés à reprendre la session ou si celle-ci doit être nettoyée automatiquement. |
| Accès aux fichiers et réunions Office365 | Session surface Hub > & alimentation | Choisissez si un utilisateur peut se connecter à Office365 pour accéder à ses fichiers et réunions. |
| Activer l’écran via des capteurs de mouvement | Session surface Hub > & alimentation | Choisissez si l’écran s’allume lorsqu’un mouvement est détecté. |
| Délai d’affichage | Session surface Hub > & alimentation | Choisissez la durée d’inactivité de l’appareil avant la désactivation de l’écran. |
| Délai d’attente de veille | Session surface Hub > & alimentation | Choisissez la durée pendant laquelle l’appareil doit être inactif avant de passer en mode veille. |
| Nom convivial | Surface Hub> À propos | Définissez le nom du SurfaceHub qui s’affiche en cas de connexion sans fil. |
| Heures de maintenance | Mise à jour et récupération&gt; WindowsUpdate&gt; Options avancées | Configurez les heures auxquelles les mises à jour peuvent être installées. |
| Configurer le serveur WindowsServer Update Services (WSUS) | Mise à jour et récupération&gt; WindowsUpdate&gt; Options avancées | Indiquez si le SurfaceHub doit recevoir les mises à jour à partir d’un serveurWSUS au lieu de WindowsUpdate. |
| Récupérer à partir du cloud | Mise à jour et sécurité&gt; Récupération | Réinstallez le système d’exploitation sur le SurfaceHub en utilisant une version d’usine à partir du cloud. |
| Enregistrer la clé BitLocker | Mise à jour et sécurité> Récupération | Sauvegardez la clé BitLocker de votre SurfaceHub sur un lecteurUSB. |
| Collecter les journaux | Mise à jour et sécurité&gt; Récupération | Enregistrez les journaux sur un lecteurUSB à envoyer à Microsoft ultérieurement. |
